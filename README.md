# Daily Focus Timer

A simple 25-minute focus timer application built with HTML, CSS, and JavaScript.

## Features

- 25-minute countdown timer
- Start and Stop buttons
- Real-time timer display (mm:ss format)
- Status indicator showing: Running / Stopped / Finished
- Clean, responsive user interface

## How to Run

1. **Download/Clone the files**: Ensure you have all the following files in the same directory:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`

2. **Open in Browser**: 
   - Double-click on `index.html` to open it in your default web browser
   - Or right-click on `index.html` and select "Open with" → your preferred browser
   - Or open your browser and use Ctrl+O (Cmd+O on Mac) to open the `index.html` file

3. **Use the Timer**:
   - Click "Start" to begin the 25-minute countdown
   - The status will show "Running" and the timer will count down every second
   - Click "Stop" to reset the timer back to 25:00 and change status to "Stopped"
   - When the timer reaches 00:00, the status automatically changes to "Finished"

## Usage Instructions

- **Start Timer**: Click the blue "Start" button to begin the countdown
- **Stop Timer**: Click the red "Stop" button to reset and stop the timer
- **Timer Display**: Shows time remaining in MM:SS format
- **Status Indicator**: 
  - Gray "Stopped" - Timer is not running
  - Green "Running" - Timer is actively counting down
  - Red "Finished" - Timer has completed the 25-minute session

## Technical Details

- **No Dependencies**: Pure HTML, CSS, and JavaScript
- **No Internet Required**: Runs completely offline
- **Browser Compatibility**: Works in all modern web browsers
- **No Installation**: Just open the HTML file in any browser
